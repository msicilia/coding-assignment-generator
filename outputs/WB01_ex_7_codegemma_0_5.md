```python
# Define a temperature value in degrees Fahrenheit
temp_fahrenheit = 77.0

# Convert Fahrenheit to Celsius
temp_celsius = (temp_fahrenheit - 32.0) * (5.0 / 9.0)

# Print the equivalent temperature in degrees Celsius
print("Equivalent temperature in Celsius:", temp_celsius)

# Convert Celsius to Fahrenheit
temp_fahrenheit_converted = (9.0 * temp_celsius) / 5.0 + 32.0

# Print the equivalent temperature in degrees Fahrenheit
print("Equivalent temperature in Fahrenheit:", temp_fahrenheit_converted)
```

**Output:**

```
Equivalent temperature in Celsius: 25.0
Equivalent temperature in Fahrenheit: 77.0
```
