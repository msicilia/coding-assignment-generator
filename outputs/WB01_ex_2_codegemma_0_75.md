```python
# Taxable income
total_income = 100000

# Personal deduction
personal_deduction = 600

# Deduction per son
deduction_per_son = 60

# Number of sons
num_sons = 2

# Taxable income
taxable_income = total_income - personal_deduction - (deduction_per_son * num_sons)

# Tax payable
tax_payable = taxable_income // 3

# Print the tax payable
print("Tax payable:", tax_payable)
```
